## 2. 데일리 리뷰

: 오늘의 수업에 대한 간단한 리뷰 및 요약을 남깁니다. 지난 수업 검색을 통해 지난 날들의 리뷰들을 검색할 수 있고 주요 간단 키워드 요약도 함께 확인 가능합니다.

### 주요 기능

- 리뷰 작성 : 오늘 하루 수업에 대한 간단한 요약 및 리뷰와 난이도 작성
- 리뷰 조회 : 조회 날짜의 리뷰를 확인할 수 있으며 요약을 주요 키워드로 빠르게 확인할 수 있습니다.
- 수업 난이도 조회 : 해당 날짜 수업에 대한 난이도를 파이 그래프로 확인할 수 있습니다.

### 기대 효과

- 수업 복습 전 리뷰를 보며 해당 수업을 회상할 수 있으며, 학습일지가 밀렸을 경우 본 리뷰를 보고 작성할 수 있습니다.

## 3. 환기 요정

: 실시간 마포구 공기 질 데이터를 분석하여 최적의 환기 시점을 알려주고, 클릭 한 번으로 팀원들에게 슬랙(Slack) 공지까지 전달하는 위치 기반 환기 의사결정 서포터입니다.

### 주요 기능

- **실시간 위치 기반 대기질 분석**: 상암동 측정소 좌표($37.58, 126.88$) 기준 PM10, PM2.5, 기온 데이터 실시간 수집
- **직관적 시각화 대시보드**:
  - 한국 표준 색상 체계 기반의 **둥근 인디케이터 바**와 현재 수치 표시선(보라색 선)
  - **Pydeck 3D 지도**를 활용한 정밀한 위치 시각화
- **지능형 환기 점수**: 기온 및 미세먼지 농도를 종합하여 0~100점 사이의 '환기 지수' 산출 및 행동 가이드 제공
- **원클릭 슬랙 연동**: 익명성을 보장하며 팀 전체에 환기 요청 메시지 즉시 발송 (성공 시 Balloon 피드백)

### 기대 효과

- **의사결정 피로도 감소**: 복잡한 수치 대신 '환기 점수'와 '행동 가이드'로 빠른 판단 가능
- **쾌적한 업무 환경**: 데이터 기반의 적기 환기를 통해 실내 공기 질 개선 및 집중도 향상
- **부드러운 커뮤니케이션**: 슬랙 자동 연동을 통해 구성원 간 불편함 없는 환기 요청 문화 정착

## 4. 좌석 랜덤 배치

: 수강생 좌석을 공정하게 랜덤 배정하고, 좌석에 대한 리뷰까지 관리하는 기능입니다. 

### 주요 기능

- DB에 등록된 **학생 목록과 좌석 정보 기반 랜덤 배정**
- 좌석별 ⭐ **별점 및 한줄 리뷰 작성 및 hover 구성**
- 좌석 선택 시 **전체 리뷰 조회 가능**

### 기대 효과

- 좌석 배정에 대한 **불필요한 갈등 최소화**
- "집중 잘 되는 자리", "피곤한 자리" 등 **좌석 특성 데이터 축적**

## 5. 카페인

하루에 마신 음료들의 카페인 함유량을 계산하여 일일 권장량 초과 여부를 알려주는 기능입니다.

### 주요기능

#### 1. 음료 선택 기반 카페인 계산

- 사용자가 마신 음료를 **다중 선택**하고 각 음료별 **섭취 잔 수**를 입력하면 음료별 카페인 함량을 기반으로 **총 섭취 카페인(mg)**을 자동 계산합니다.

#### 2. 일일 권장량 대비 섭취 현황 시각화

- 성인 기준 **일일 카페인 권장량(400mg)** 대비 현재 섭취량을 시각화하여 안전 / 주의 / 초과 구간을 직관적으로 확인할 수 있습니다.
- 현재 섭취량 수치를 **mg 단위로 명확하게 표시**합니다.

#### 3. 텍스트 기반 섭취 현황 요약

- 그래프와 별도로 현재 총 섭취량, 일일 권장량, 남은 섭취 가능량을 **텍스트로 한 번 더 요약**하여 수치 중심의 확인이 가능합니다.

#### 4. 음료별 카페인 비중 시각화

- 선택한 음료들이 전체 섭취 카페인에서 차지하는 비율을 **도넛 형태의 파이 차트**로 표현하고 카페인 소비 흐름을 보여줍니다.

#### 5. 섭취 상태에 따른 즉각적인 피드백

- 총 카페인 섭취량에 따라 상태 메시지를 실시간으로 제공해 **사용자의 섭취 행동을 자연스럽게 유도**합니다.
    
    ✅ 안전, ⚠️ 주의, 🚨 초과
    

### 기대효과

- 하루 섭취한 음료를 입력함으로써 현재 섭취한 카페인 총량을 파악할 수 있다.
- 권장량 초과 여부를 시각화로 제공해 카페인 초과 섭취를 지양한다.
