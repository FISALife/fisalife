## 1. 메인 페이지

서비스의 첫 진입 화면으로,  
각 하위 기능에 대한 **간단한 설명과 안내**를 제공하여

---

## 2. 데일리 리뷰

오늘의 수업을 **한 줄로 기록하고 공유**하는 학습 회고 기능입니다.  
날짜별 검색과 키워드 요약을 통해 지난 수업을 빠르게 되돌아볼 수 있습니다.

### 주요 기능
- **리뷰 작성**: 수업 요약 및 난이도 기록
- **리뷰 조회**: 날짜별 리뷰 및 주요 키워드 확인
- **난이도 시각화**: 수업 난이도 분포를 파이 그래프로 제공

### 기대 효과
- 수업 복습 전 빠른 회고 가능
- 학습일지가 밀렸을 때 기록 보조 역할 수행

---

## 3. 환기 요정

실시간 마포구 공기질 데이터를 기반으로  
**지금 환기해도 괜찮은지 판단을 도와주는 위치 기반 환기 서포터**입니다.

### 주요 기능
- 상암동 측정소 기준 **PM10 / PM2.5 / 기온 실시간 수집**
- 한국 표준 색상 체계 기반 **직관적 시각화 대시보드**
- 기온·미세먼지를 종합한 **환기 점수(0~100점) 산출**
- **원클릭 Slack 연동**을 통한 익명 환기 요청 전송

### 기대 효과
- 수치 해석 부담 없이 빠른 의사결정
- 데이터 기반 환기로 실내 환경 개선
- 눈치 보지 않는 환기 요청 문화 형성

## 4. 좌석 랜덤 배치

좌석을 **공정하게 랜덤 배정**하고  
좌석별 경험을 리뷰로 관리하는 기능입니다.

### 주요 기능
- 학생·좌석 DB 기반 랜덤 배정
- 좌석별 ⭐ 별점 및 한 줄 리뷰
- 좌석 선택 시 전체 리뷰 조회

### 기대 효과
- 좌석 배정에 대한 불필요한 갈등 감소
- 좌석 특성 데이터 축적 및 공유

---

## 5. 카페인 계산기

하루 동안 마신 음료의 카페인 섭취량을 계산하고  
**일일 권장량 초과 여부를 시각적으로 안내**합니다.

### 주요 기능
- 음료 선택 + 섭취 잔 수 입력 기반 총 카페인 계산
- 성인 기준 **일일 권장량(400mg)** 대비 시각화
- 음료별 카페인 비중 도넛 차트 제공
- 섭취량에 따른 상태 메시지 (✅ 안전 / ⚠️ 주의 / 🚨 초과)

### 기대 효과
- 무의식적인 카페인 과다 섭취 방지
- 건강한 음료 섭취 습관 형성

수강생들이 직접 유용한 링크를 공유하는  
**공동 정보 아카이브 페이지**입니다.

### 주요 기능
- 카테고리별 링크 관리 (공모전, 코테, AI 이슈 등)
- 링크 + 설명 직접 등록
- DB 저장 기반 데이터 유지
- 탭 UI로 가독성 강화

### 기대 효과
- 정보 손실 방지
- 집단 지식의 지속적 축적
- 프로젝트·학습 시 빠른 레퍼런스 접근
